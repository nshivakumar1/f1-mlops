44 Mistakes. Every One Documented.
The failures that made this system production-ready — and what I'd tell myself on day one
🔴 Security
Live API Key Committed to Git
What happened
A New Relic license key was inlined in a CLI command saved to .claude/settings.json and committed. GitGuardian flagged it in minutes.
Fix
Rotated key immediately. Rewrote git history with --amend + force push. Added pattern to .gitignore.
🟡 AWS CLI
Updating One Env Var Wipes All Others
What happened
update-function-configuration --environment Variables={KEY=val} replaces the entire env map. SageMaker endpoint, S3 bucket, Sentry DSN — all gone mid-race.
Fix
Always read current env vars first, merge, then write back. Or use EventBridge target Input JSON for race-day overrides.
🔵 API Design
66 API Calls Per Cycle Hit Rate Limits
What happened
Called OpenF1 once per driver per endpoint: 22 drivers × 3 endpoints = 66 requests. OpenF1 enforces 60 req/min. Every invocation failed.
Fix
Fetch all drivers in one call, group by driver_number in Python. 66 calls → 3 calls per invocation.
🟡 Terraform
templatefile() Breaks on %{ in Scripts
What happened
Elasticsearch index patterns like %{+YYYY.MM.dd} and Logstash grok patterns like %{DATA:field} — Terraform treats %{ as a template directive.
Fix
Escape with %%{ in every .tpl file. Terraform outputs a literal %{ at runtime. Every single one.
🔵 OpenF1
Latest Session Was Abu Dhabi in March
What happened
OpenF1 pre-populates the full season calendar. sessions[-1] picked the last entry chronologically — Abu Dhabi in December — during the Australian GP.
Fix
Filter to date_start <= datetime.utcnow() before picking the last session. "Latest in dataset" ≠ "latest relative to now".
🔴 CI/CD
Terraform Rezips Lambda Without Dependencies
What happened
archive_file re-runs during terraform apply in the deploy job — no pip packages installed. Enrichment Lambda became a 9KB source-only ZIP. No module named 'groq'.
Fix
Build manylinux ZIPs in plan job, pass as artifacts, redeploy from pre-built ZIPs after terraform apply.

The Real Lesson
Documentation is the product. The 44-entry mistake log outlasted every sprint plan and architecture diagram I drew.
Break in production, fix in production. Race-day pressure forces decisions that tutorials never simulate. That's where the real learning happens.
Errors that are caught silently are the worst kind. wget -q, swallowed exceptions, missing capture_exception() calls — silence kills faster than noise.
